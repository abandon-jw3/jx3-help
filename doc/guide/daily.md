# 日常活动查询

查询服务器日常活动、维护公告、官方新闻等信息。

## 日常

查询今日服务器活动日历，包括秘境大战、战场任务、宗门事件等。

**指令格式**：`日常 [服务器]`

**别名**：`每日`

**示例**：
- `日常` - 查询绑定区服的日常
- `日常 唯我独尊` - 查询指定区服的日常

**效果展示**：

![日常查询](/images/daily-calendar.png)
