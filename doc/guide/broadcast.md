# 播报订阅

::: warning 权限说明
以下功能开关仅限**群主**或**管理员**使用。开启后，机器人会自动推送对应类型的游戏事件到群聊。
:::

::: tip 提示
播报功能会根据群绑定的区服进行过滤，只推送对应区服的事件。请确保已正确[绑定区服](/guide/getting-started#绑定区服)。
:::

## 开服监控

接收服务器开服/维护状态变更通知。

**开启**：`开启开服监控` 或 `开启开服`

**关闭**：`关闭开服监控` 或 `关闭开服`


---

## 新闻播报

**开启**：`开启新闻播报` 或 `开启新闻`

**关闭**：`关闭新闻播报` 或 `关闭新闻`
