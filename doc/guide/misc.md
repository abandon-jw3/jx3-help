# 其他功能

一些实用的辅助功能。

## 科举

查询科举考试的答案。

**指令格式**：`科举 [题目关键字]`

**示例**：
- `科举 李白`
- `科举 长安`

**效果展示**：

![科举答案](/images/exam-answer.png)
